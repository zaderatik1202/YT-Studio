Your Free Production System — Setup Guide

Follow these steps once to set up your complete free team system. Everything runs on Google Workspace (free tier) — no paid tools needed.

1
📁 Set Up Google Drive Structure
Create a shared folder called "YT Studio" in your Drive. Share it with Nayan & Guddu. Create the folder structure shown below.
2
📊 Create the Tracker Sheet
Copy the Google Sheet template link below. Share with team. This is your single source of truth — every video has a row, every task has a status cell.
3
🔔 Set Up Auto Notifications
In the Sheet → Tools → Notification Rules → "Any changes made" → Email notification. Each person sets this on their own account.
4
📂 One Folder Per Video
For each video, create a numbered folder in Drive: "V01 - Video Title". Inside it, create subfolders for each stage (Script, Images, Voice, Video).
5
📋 Workflow Rules
You add video to Sheet → write script → Nayan generates images → Guddu generates video → You do voice + edit. Update status in Sheet after each step.
6
📌 Bookmark This Dashboard
Save this HTML file to your Drive too! It's fully offline. Or host it free on GitHub Pages so all 3 of you can open it anytime from any device.
